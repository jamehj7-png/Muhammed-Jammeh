# Home SOC Lab

## Project Overview
This project is a personal Security Operations Center (SOC) lab built to simulate a small enterprise network environment. The goal is to practice system administration, network monitoring, and cybersecurity detection techniques.

## Objectives
- Build a virtual enterprise environment
- Simulate common cyber attacks
- Monitor system logs and network activity
- Practice detecting suspicious behavior
- Improve incident response skills

## Lab Environment

### Virtual Machines
- Windows Server (Domain Controller)
- Windows Client
- Kali Linux (Attacker Machine)

### Technologies Used
- Active Directory
- Windows Server Administration
- Linux
- Virtualization (VirtualBox / VMware)
- SIEM tools (Wazuh planned)
- Networking (TCP/IP)

## Planned Security Scenarios
- Brute force login attempts
- Password spraying attacks
- Privilege escalation simulation
- Log monitoring and detection
- Security hardening

## Learning Goals
This lab helps strengthen practical skills in:

- System administration
- Network infrastructure
- Cybersecurity monitoring
- Blue team security operations
